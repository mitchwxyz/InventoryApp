# What?
A Basic CRUD App.  Built using htmx, FastAPI, and MongoDB.

### Core Functions
- Create, Delete, and Edit Records from the database
- user_id cookie shows your records only

### UI Features (htmx)
- Active Search
- Infinite Scroll
